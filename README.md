# Bn technologies - Kenya Billing System (Render-ready)

This project is prepared for deployment on Render with Bootstrap 5 and basic role support (Admin, Cashier).

## What's included
- Django project configured for production (environment-driven settings)
- Core app with Customers, Products, Invoices, Payments, InvoiceItem
- Role handling: creates 'Admin' and 'Cashier' groups automatically on first dashboard load
- Bootstrap 5 UI with header 'Bn technologies'
- M-Pesa STK push demo and callback endpoint (sandbox)
- Procfile and runtime.txt for Render
